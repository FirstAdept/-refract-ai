О проекте

FITBODY — это комплексное решение для людей, которые хотят взять под контроль своё здоровье и форму. Мы построили платформу, где AI-ассистент анализирует питание, тренировки и прогресс, давая рекомендации в реальном времени.

Ключевая фишка — сканер штрихкодов с распознаванием продуктов через OpenFoodFacts и кастомный ML-движок для расчёта калорий по фото блюда. Пользователь просто фотографирует тарелку — приложение само определяет состав и БЖУ.

Что мы сделали

  • Mobile-first приложение на React Native (iOS + Android)
  • Веб-версия личного кабинета на React + TypeScript
  • Backend на Node.js с PostgreSQL и Redis
  • AI-интеграция с OpenAI для персональных рекомендаций
  • Интеграция с базой OpenFoodFacts (3M+ продуктов)
  • Платёжная система с подписками и trial-периодом
  • Telegram-бот для быстрого учёта калорий

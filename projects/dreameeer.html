О проекте

DREAMEEER переосмысливает социальные сети — вместо ленты с инфошумом пользователи создают доски своих мечт и целей, делятся прогрессом, находят единомышленников.

Каждая мечта оживает через интерактивную визуализацию: можно прикрепить фото, видео, аудио, поделиться этапами достижения. Сообщество поддерживает реакциями и историями вдохновения.

Что мы сделали

  • Веб-приложение на React + TypeScript + Vite
  • Мобильная версия на React Native
  • Backend на Node.js с GraphQL API
  • Интерактивные доски мечт с drag-and-drop
  • Система реакций, комментариев, подписок
  • Real-time уведомления через WebSocket
  • Загрузка медиа в S3 с оптимизацией
